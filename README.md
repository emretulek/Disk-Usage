# Disk Uage Widget Plugin

This project was developed as a disk usage widget for windows widgets.

### Internal Settings

```json
{
  "usage_font_size": 24.0,
  "usage_font_size2": 14.0,
  "usage_foreground": "#FFBC8F8F",
  "bar_color": "#FF8B0000",
  "bar_background": "#FFBC8F8F",
  "bar_height": 5,
  "drivers": [
    "C",
    "D"
  ]
}
```
